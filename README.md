# Sales Performance & Customer Insights Dashboard

## Tools Used
Microsoft Excel | Pivot Tables | Power Query | Slicers

## Project Overview
Built an interactive Excel dashboard tracking sales performance across 
product categories, cities, and time periods using 9,994 transactions 
from a retail superstore dataset.

## Key Metrics
- Total Revenue: $2,297,200
- Total Profit: $286,397
- Total Quantity Sold: 37,873
- Total Transactions: 9,994

## What the Dashboard Shows
- Sales by Category (Furniture, Office Supplies, Technology)
- Top 10 Cities by Revenue
- Monthly Revenue Trend
- Interactive slicers for Region and Category filtering

## Key Findings
- Technology had the highest sales at $836,154
- New York City was the top revenue generating city
- Office Supplies had the highest profit margin

## Dataset
Sample Superstore dataset — 9,994 rows of retail transaction data
